\## 🛠️ 技术架构

\### 核心能力组合

| 能力 | 技术定位 | 病房核心用途 |

|------|----------|--------------|

| VLM视觉理解 | 非接触式监测 | 患者行为/生命体征识别、环境状态监测、护理合规判断 |

| ASR语音识别 | 自然交互入口 | 患者/医护语音指令采集、虚弱患者无接触呼叫 |

| TTS语音合成 | 交互反馈出口 | 患者康复提示、医护异常警报、语音交互回答 |

| 文生图 | 可视化输出 | 个性化康复指导图、护理报告可视化、异常事件记录 |



\### 覆盖场景

✅ 患者行为与生命体征监测 ✅ 医患自然语言交互

✅ 护理任务智能提醒 ✅ 患者教育与康复指导

✅ 病房环境智能管理 ✅ 多模态患者报告生成



\## 📂 工程说明

\- `index.html`：主Demo页面，包含所有交互功能、方案内容、模拟演示

\- `assets/css/style.css`：医疗系统专业配色，适配移动端/PC端

\- `assets/js/main.js`：全交互逻辑，含模拟演示、锚点跳转、动态展开等功能

\- `docs/solution-detail.md`：完整方案文字版，可作为评审附件
